MEMO GIT ET GITHUB


Ce repository est un mémo des commandes de base à connaître 
pour utiliser Git et GitHub.

Le fichier commandes_terminal.txt regroupe les commandes 
principales pour utiliser la console.

Le fichier commandes_commit_git.txt reprend les commandes 
de base à connaître pour pouvoir utiliser Git, ajouter des 
fichiers à l'index ou les modifier, et faire des commits.

Le fichier commandes_envoyer_recuperer_modifications.txt 
regroupe les commandes principales pour synchroniser son code 
local avec ses repos en ligne.
